## Edge cases

### Persistent Views
Критически важно: после перезапуска бота кнопки должны продолжать работать.
- Все persistent views регистрируются в `setup_hook` через `bot.add_view()`
- Все `custom_id` статичные (не динамические)
- Тест: перезапустить бота, нажать старую кнопку

### Таймауты
- Пользователь нажимает кнопку через 15+ минут на ephemeral view
- Обработка: `interaction.response.send_message("Меню устарело, используйте /start", ephemeral=True)`

### Изоляция
- Два пользователя одновременно в меню — состояние не пересекается
- user_id в custom_id или проверка interaction.user

### Ошибки отправки
- Канал удалён → graceful error, лог, не крашить задачу
- Пользователь заблокировал DM → try/except, уведомить admin
- Rate limits → asyncio.sleep(0.5) между отправками
